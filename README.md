## 📱 Handy-Nutzung

1. Streamlit Cloud URL öffnen
2. iOS: Teilen → Zum Home-Bildschirm
3. Android: Menü → Zum Startbildschirm

## ⚙️ Setup

1. Repository klonen
2. `pip install -r requirements.txt`
3. `.env.example` zu `.env` kopieren und Keys eintragen
4. `python -m streamlit run app.py`

## ☁️ Streamlit Cloud

API-Keys unter: Settings → Secrets

```toml
OPENROUTER_API_KEY = "sk-or-v1-..."
OPENAI_API_KEY = "sk-proj-..."
```
